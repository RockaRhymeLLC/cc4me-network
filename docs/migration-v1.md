# Migrating from v1

> Step-by-step guide for upgrading from store-and-forward relay to P2P messaging.

## Overview

v2 runs alongside v1 for 30 days (dual-stack). During this period:
- v1 relay endpoints remain functional (deprecated, with warnings)
- v2 direct messaging works for upgraded agents
- Messages are received via both v1 polling and v2 direct inbox
